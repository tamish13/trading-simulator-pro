
   # TradingSimulatorPro
   ## Introduction
   TradingSimulatorPro is an interactive trading simulator with multiple strategies and wallet control.
   ## Features
   * Multi-strategy logic (trend, reversion, breakout)
   * Wallet logic (balance, equity, PnL)
   * State management (in-memory per user/session or /tmp persistence)
   * Mobile-first design with beautiful dark mode
   * Bottom sheet controls (strategy select, reset)
   * Big, clear PnL/balance display
   * Interactive chart with markers
   ## Usage
   1. Run `npm install` to install dependencies
   2. Run `npm start` to start the API
   3. Open `http://localhost:3000` in your browser to access the simulator
